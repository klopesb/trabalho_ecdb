# 📊 Análise de Dados Biológicos | Grupo 5
## 🧬 RNA-Seq - Uterine Corpus Endometrial Carcinoma

📘 Trabalho Prático da UC de Extração de Conhecimento de Dados Biológicos (2024/2025)

Este repositório contém a análise de um conjunto de dados clínicos e tumorais do estudo UCEC TCGA PanCancer Atlas (2018), disponibilizado pelo cBioPortal — uma plataforma interativa para explorar dados de câncer.

O dataset utilizado refere-se ao carcinoma endometrial uterino e faz parte do consórcio The Cancer Genome Atlas (TCGA), que reúne dados genômicos e clínicos de milhares de pacientes oncológicos.

O projeto é dividido nas seguintes partes:

### 1ª Parte: 

- 📚 Descrição do dataset: origem, características clínicas e biológicas, relevância biomédica

- 🧹 Pré-processamento e limpeza de dados

- 📈 Sumarização e visualização: estatísticas descritivas, gráficos, histogramas e densidades

- 🧪 Testes estatísticos univariados

- 🎯 Análise de expressão diferencial e enriquecimento funcional

### 2ª Parte:

- 🧬 Clustering de genes e/ou amostras (agrupamento não supervisionado)

- 📉 Redução de dimensionalidade: MDS, PCA ou métodos semelhantes

- 🤖 Modelos preditivos com Machine Learning: classificação, regressão e comparação entre modelos

- 🌟 Seleção e importância de genes: identificação de biomarcadores relevantes
